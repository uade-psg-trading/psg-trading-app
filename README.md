## Development

Correr `npm i` (`yarn install` || `pnpm i`. Depende cual usen) y luego ejecutar:

```bash
npm run dev

# O para que habra directamente el browser
npm run dev --open
```

## Build

Para crear una build productiva simplemente correr:

```bash
npm run build
```
